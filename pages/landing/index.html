How to use Cement

First, download the Cement installer, and then run it. Once Cement is installed you can then start using Cement mods.
To use a Cement mod, download any Cement file (a file that has the '.cmt' file extension), then locate the folder where Gang Beasts is installed, and copy the Cement file into the 'Mods' directory in your Gang Beasts folder. If you don't have a 'Mods' folder, make sure Cement is installed, then run Gang Beasts and close it, and a 'Mods' folder should appear.

About Cement

Cement is a mod loader built for Gang Beasts using BepInEx, written in the C# Language, and is what holds users' mods together.

Cement provides helpful modding tools, making it easy for creators to make mods, and users to play them.
